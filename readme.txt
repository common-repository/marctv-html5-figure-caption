Requires at least: 3.0

Stable tag: 1.2

== Description ==

Replaces image markup with the HTML5 figure and figcaption tags. Removes the inline styles, too. 

== Installation ==

Upload the "marctv-html-figure-caption" folder into the plugins folder of your blog, Activate it. You're done!

= Why did you do this? =

Because needed a simple plugin to do this job and decided to do it on my own.
